# 📌 Project Overview

MITILENCE 2K26 is a **college technical symposium website** designed to manage:

- Event details
- Participant registration
- Team management
- Paper submission
- Admin approval system
- Event gallery
- AI assistant support

This platform helps organizers **efficiently manage event participants and registrations**.

---

# 🎯 Features

### 🧾 Event Management
- Technical & Non-technical event listings
- Detailed rules and instructions
- Team size configuration

### 👥 Registration System
- Individual and team registration
- Paper upload (Google Drive integration)
- Offline payment model

### 📊 Admin Dashboard
- Approve / reject registrations
- Disqualify teams
- Manage participants
- View uploaded research papers

### 🖼 Gallery System
- Previous year event photos
- Current year event media

### 🤖 AI Chat Assistant
- Event guidance
- Participant support

### 📱 Responsive Design
- Mobile friendly
- Clean navigation
- Fast performance
